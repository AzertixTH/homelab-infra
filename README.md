🧠 Homelab Infrastructure

Personal homelab built to learn and manage real-world infrastructure concepts in a practical, hands-on environment.

🔧 Setup

• Multiple nodes (Ubuntu Server)
• SSH-based remote management
• Reverse proxy (nginx)
• Containerized services (Docker)
• Dedicated AI server (RIAS)

🖥️ Nodes

• Bifröst → Infrastructure node (nginx, dashboard, networking)
• RIAS → AI server and backend services
• Midgard → Home automation (planned)

🌐 Features

• Centralized access via reverse proxy
• Service separation across multiple nodes
• Full remote management via SSH
🧭 Architecture
User → Bifröst (nginx) → Internal services (RIAS / Midgard)

🎯 Goal

To simulate a small-scale production environment and develop practical skills in system administration, networking, and infrastructure management.
